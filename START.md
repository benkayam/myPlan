# 🚀 איך להתחיל

## הפעלה מהירה

### אופן 1: דפדפן ישיר (פשוט)
1. פתח את הקובץ `index.html` בדפדפן (לחיצה כפולה)
2. **אבל**: יכול להיות שהדפדפן יחסום את SheetJS בגלל CORS

### אופן 2: שרת מקומי (מומלץ) ✅

#### עם Python (מותקן ברוב המחשבים):
```bash
cd /Users/benkayam/Desktop/projects/myPlan
python3 -m http.server 8000
```

אז פתח בדפדפן:
```
http://localhost:8000
```

#### עם Node.js:
```bash
npx http-server -p 8000
```

#### עם PHP:
```bash
php -S localhost:8000
```

## 🎯 מה לעשות אחרי שהשרת רץ

1. **פתח את הדפדפן** ב-`http://localhost:8000`
2. **העלה את הקובץ** `myPlan.xlsx` (או כל קובץ Excel אחר)
3. **המתן** שהמערכת תנתח את הקובץ
4. **תהנה** מהדשבורד! 🎉
